# Changelog

## Unreleased

- 新增统一 `CommandReview`，使计划展示、执行前复检、验证和评测共享同一套命令事实与安全决策。
- 安全审查记录全部命中规则；执行前风险提高时旧授权失效，批处理不会静默执行。
- 修复文件候选替换后丢失任务意图、实体和模型风险建议的问题。

## 0.2.0

- 增加结构化意图、实体、模型风险建议和受限组合命令分析。
- 统一模型请求、历史与日志的敏感信息脱敏。
- 增加 Docker 沙箱执行后端、错误分类和 200 条系统评测集。
- 增加文件覆盖检测、增强解释和多维评测报告。

## 0.1.0

- 完成计划、安全决策、受控执行、结果验证和历史审计闭环。
- 支持交互、单次任务、JSON、批处理、本地模型和 DeepSeek。
- 增加跨平台测试、进程树超时终止和输出截断。
